#Статусы платежного поручения

----------

**Назначение справочника**: справочник предназначен для хранения статусов платежных поручений. 

**Заполнение бланка**:

Справочник является предзаполненным. 

![](topic:НСИ.AddFiles.Screenshot_1626.jpg)

Для добавления новой записи откройте картотеку нажмите кнопку **Добавить** ![](topic:НСИ.AddFiles.Btn_Add.png) и заполните открывшуюся форму.

Заполните следующие поля вручную: 

- Название - наименование статуса платежного поручения; 
- Код – Аналитический код редактируемого или вновь создаваемой записи. Новый код не должен совпадать ни с одним из кодов ранее введенных. 
Для редактирования существующей записи в картотеке воспользуйтесь кнопкой "Редактировать" ![](topic:НСИ.AddFiles.Btn_Edit.png)  .

Для ввода в действие введенной информации нажмите на кнопку Применить ![](topic:НСИ.AddFiles.Btn_OK.png) . Для закрытия справочника - Отмена ![](topic:НСИ.AddFiles.BtnCloseCancel.png) .
