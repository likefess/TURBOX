# Создание нового класса реквизита

- Описываем Rec-класс и Struct-классы для каждой подтаблицы в Rec.Properties
  (см. например, Rec.Properties.IServiceBuildReport)
- Описываем Frm-класс для каждой закладки с дополнительными настройками
  (см. например, Frm.Properties.IServiceBuildReport)
- В Rec/Struct классах описываем все дополнительные поля, с которыми работает
  этот реквизит, а также переменные с описанием метода их расчета. Устанавливаем
  атрибут AutoGenerate=True для тех полей / переменных, которые будут использоваться
  при автогенерации.

## Методы Rec-класса
- LoadMoreSettings - подзагрузка и настройка дополнительных закладок с настройками
- SetupForm - настройка контролов (видимость, доступность и т.п.) на дополнительных
  формах в зависимости от других настроек реквизита
- SetCurrentPage - установка текущей закладки при открытии формы
- InitService и Run в наследнике IService - методы инициализации и запуска сервиса.
- EnabledWithLockByDefault - доступность при установленном замке
- ShowProperty - создание контрола на основании реквизита (для сервисов не требуется)
- CrfStorage - создание Storage для генерации crf-файла (для сервисов не требуется)
- GenerateCod - создание строки для генерации cod-файла (для сервисов не требуется)
- SetDefault - установка значений по-умолчанию
