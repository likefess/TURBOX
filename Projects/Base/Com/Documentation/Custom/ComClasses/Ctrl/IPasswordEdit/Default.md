# Ctrl.IPasswordEdit

Класс *Ctrl.IPasswordEdit* предназначен для работы с полями ввода паролей.
В полях ввода такого типа все символы выводятся звездочками (*).

Наследует все свойства и методы родительского класса Ctrl.IEdit.
