# Процедура ChangeValue

## Описание

    proc ChangeValue (aValue :Variant; aIndex :Integer);

## Параметры

**aValue** - новое значение.

**aIndex** - номер строки, если переключатель лежит в таблице.

## Назначение

Изменение значения переключателя.
