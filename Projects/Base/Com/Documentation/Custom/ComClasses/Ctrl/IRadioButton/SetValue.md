# Процедура SetValue

## Описание

    proc SetValue (aValue :Variant; {aIndex :Integer});

## Параметры

**aValue** - новое значение.

**aIndex** - номер строки, если переключатель лежит в таблице.

## Назначение

Установка значения переключателя.
