# Процедура LoadHtml

## Описание

    proc LoadHtml (aHtml :String);

## Параметры

**aHtml** - загружаемый фрагмент html.

## Назначение

Загрузка фрагмента html и внедрение его в конец тега *body* js-скрипта.
