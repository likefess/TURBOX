# Процедура InsertText

## Описание

    proc InsertText(aIndex :Integer; aText :String);

## Параметры

**aIndex** - индекс места вставки.

**aText** - текст.

## Назначение

Вставка текста **aText** в указанное место по заданному индексу **aIndex** .
