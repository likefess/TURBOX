# Процедура SetValue

## Описание

    proc SetValue(aValue :Variant; {aIndex :Integer});

## Параметры

**aValue** - устанавливаемое значение.

**aIndex** - номер строки, если флаг - столбец таблицы.

## Назначение

Установка значения флага.
