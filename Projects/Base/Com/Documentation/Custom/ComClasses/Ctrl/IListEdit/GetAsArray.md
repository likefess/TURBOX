# Функция GetAsArray

## Описание

    func GetAsArray({aIndex :Integer}) :String[];

## Параметры

**aIndex** - номер строки, если контрол лежит в таблице. По умолчанию 1.

## Назначение

Получение списка возможных значений в виде массива строк.
