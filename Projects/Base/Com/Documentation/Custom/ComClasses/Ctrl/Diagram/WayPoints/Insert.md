# Процедура Insert

## Описание

    Insert (aIndex :Integer; aX :Numeric; aY :Numeric);

## Параметры

**aIndex** - индекс.

**aX** - координата по оси X.

**aY** - координата по оси Y.

## Назначение

Добавление точки на указанную позицию.
