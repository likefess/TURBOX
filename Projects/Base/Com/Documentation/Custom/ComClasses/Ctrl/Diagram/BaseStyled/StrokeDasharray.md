# Поле StrokeDasharray

## Описание

    StrokeDasharray :String;

## Назначение

Шаблон обводки (бордюра) элемента.

Задаётся в формате x,y{,x,y} (x - размер штриха в пикселях, y - разрыв в пикселях).
Пустое значение соответствует сплошной линии (1,0).
