# Поле TableRowsCount

## Описание

    TableRowsCount :Integer;

## Назначение

Свойство содержит количество строк в таблице.

Имеет смысл только для табличного фрейма.

## <span style="color:red">Внимание</span>

Свойство устарело.

Для вывода таблиц используйте [ITable](topic:.Custom.ComClasses.Ctrl.ITable.Default).
