# Поле MarginsEx

## Описание

    MarginsEx :Integer[];

## Назначение

Отступы для сторон фрейма. Величины отступа измеряются в пикселях. В поле хранится массив из четырех
элементов, задающих величину отступа слева, сверху, справа, снизу.
