# Процедура EachSubframe

## Описание

    proc EachSubframe(aProc :proc (aFrame :Ctrl.IFrame));

## Параметры

**aProc** - процедура, обрабатывающая дочерние фреймы.

* **aFrame** - обрабатываемый фрейм.

## Назначение

Применение процедуры **aProc** к каждому субфрейму.
