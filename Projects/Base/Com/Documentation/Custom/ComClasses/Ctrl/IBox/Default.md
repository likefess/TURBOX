# Ctrl.IBox

Класс *Ctrl.IBox* предназначен для работы с боксами.

Бокс - это фрейм, который имеет заголовок и может быть свёрнут.

Наследует все свойства и методы родительского класса Ctrl.IFrame.

Непосредственно в классе *Ctrl.IBox* определены следующие свойства класса:
* [Поле CollapseSource](CollapseSource)
* [Функция CreateBox](CreateBox)

и свойства объекта:
* [Поле ButtonHelp](ButtonHelp)
* [Поле Title](Title)
* [Поле Collapsed](Collapsed)
* [Поле ContentMarginsEx](ContentMarginsEx)
* [Событие OnVerifyCollapsed](OnVerifyCollapsed)
* [Событие OnChangeCollapsed](OnChangeCollapsed)
* [Процедура SaveState](SaveState)
