# Процедура ImageFromString

## Описание

    proc ImageFromString(const aFormat :String; const aData :String);

## Параметры

**aFormat** - формат изображения. Возможные значения: 'bmp', 'jpg', 'ico', 'wmf' и 'emf'.

**aData** - данные.

## Назначение

Запись в [Field](Field) картинки из строки.
