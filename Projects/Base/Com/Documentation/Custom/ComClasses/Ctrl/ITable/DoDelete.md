# Процедура DoDelete

## Описание

    proc DoDelete(aIndex :Integer);

## Параметры

**aIndex** - номер строки.

## Назначение

Действие при удалении строки таблицы.

## <span style="color:red">Внимание</span>

Будет удален в 10.6. Используйте метод [Delete](topic:.Custom.ComClasses.Ctrl.ITable.Delete).
