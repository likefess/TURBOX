# Функция CreateEx

## Описание

    func CreateEx({aName :String}; aWidth :Numeric; {aCaption :String}) :Ctrl.IStatic;

## Параметры

**aName** - имя создаваемого статика.

**aWidth** - ширина.

**aCaption** - текст.

## Назначение

Создание статика с заданным именем, шириной и текстом.
