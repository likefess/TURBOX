# Процедура DeleteRows

## Описание

    proc DeleteRows(aIndex :Integer; {aCount :Integer});

## Параметры

**aIndex** - позиция начала удаления.

**aCount** - количество строк, по умолчанию 1.

## Назначение

Удаление строк.
