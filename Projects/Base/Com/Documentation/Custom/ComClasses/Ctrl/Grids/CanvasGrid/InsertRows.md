# Процедура InsertRows

## Описание

    proc InsertRows(aIndex :Integer; {aCount :Integer});

## Параметры

**aIndex** - позиция вставки.

**aCount** - количество строк, по умолчанию 1.

## Назначение

Вставка строк.
