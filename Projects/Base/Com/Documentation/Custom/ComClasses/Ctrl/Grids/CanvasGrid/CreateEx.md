# Функция CreateEx

## Описание

    func CreateEx(aCols :Integer; aRows :Integer; {aClusterWidth :Integer}; {aClusterHeight :Integer}) :Ctrl.Grids.CanvasGrid;

## Параметры

**aCols** - количество столбцов.

**aClusterHeight** - высота кластера.
