# Процедура InsertButton

## Описание

    proc InsertButton(aButton :Ctrl.IButton; {iPos :Integer});

## Параметры

**aButton** - кнопка, объект класса [Ctrl.IButton](topic:.Custom.ComClasses.Ctrl.IButton.Default).

**iPos** - позиция, на которую происходит вставка. По умолчанию 1.

## Назначение

Вставка новой кнопки-команды перед кнопкой-командой номер **iPos**.
