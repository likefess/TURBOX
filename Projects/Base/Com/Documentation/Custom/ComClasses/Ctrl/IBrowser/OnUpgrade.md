# Событие OnUpgrade

## Описание

    OnUpgrade :proc (aSender :Ctrl.IBrowser; var Src :String);

## Параметры

**aSender** - браузер, с которым произошло событие.

**Src** - источник обновления.

## Назначение

Обработчик события, которое происходит при обновлении страницы.
