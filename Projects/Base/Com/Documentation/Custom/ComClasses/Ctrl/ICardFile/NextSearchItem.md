# Процедура NextSearchItem

## Описание

    proc NextSearchItem(aSearchRole :Integer);

## Параметры

**aSearchRole** - правило перехода:

* 1 - первый результат;
* 2 - предыдущий;
* 3 - следующий;
* 4 - последний.

## Назначение

Переход к другому результату поиска.
