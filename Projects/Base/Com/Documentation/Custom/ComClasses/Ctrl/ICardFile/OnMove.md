# Событие OnMove

## Описание

    OnMove :proc(aSender :Ctrl.ICardFile; aRec :Rec.IRecord);

## Параметры

**aSender** - картотека, в которой происходит событие.

**aRec** - перемещаемая запись.

## Назначение

Обработчик события, которое происходит при перемещении записи в картотеке.
