# Функция RowsCount

## Описание

    RowsCount ({aOnlyVisible :Logical}) :Integer;

## Аргументы

**aOnlyVisible** - учитывать только видимые строки. По умолчанию равно False.

## Назначение

Получение количества строк в контроле.
