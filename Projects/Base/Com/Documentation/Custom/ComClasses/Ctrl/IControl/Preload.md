# Процедура Preload

## Описание

    proc Preload (aRedraw :Logical);

## Аргументы

**aRedraw** - если равно True, фрейм перерисовывается.

## Назначение

Предварительное вычисление состояния элемента управления (контрола) при выгрузке на веб-клиент.
