# Функция GetInheritedPopupMenu

## Описание

    GetInheritedPopupMenu :Ctrl.IPopupMenu;

## Назначение

Получение контекстного меню таблицы, в которой лежит контрол.

Если контрол не лежит в таблице, возвращает nil.
