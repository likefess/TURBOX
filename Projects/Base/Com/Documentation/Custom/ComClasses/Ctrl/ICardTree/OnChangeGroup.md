# Событие OnChangeGroup

## Описание

    OnChangeGroup :proc(aGroup :Record);

## Параметры

**aGroup** - ссылка на запись той группы, в которую осуществляется переход.

## Назначение

Обработчик события, которое происходит при переходе из одной группы в другую.
