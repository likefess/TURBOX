# Поле CanDesignate

## Описание

    CanDesignate :Logical;

## Назначение

Данное поле принимает значение True, если поле [LookupField](topic:.Custom.ComClasses.Ctrl.IEdit.LookupField)
содержит значение числового, целого, логического типа или типа даты.

*не используется*
