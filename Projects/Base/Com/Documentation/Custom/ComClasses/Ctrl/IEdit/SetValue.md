# Процедура SetValue

## Описание

    proc SetValue(aValue :Variant; aIndex :Integer = 1);

## Параметры

**aValue** - устанавливаемое значение;

**aIndex** - номер строки, если поле - столбец таблицы.

## Назначение

Установка значения в поле ввода.
