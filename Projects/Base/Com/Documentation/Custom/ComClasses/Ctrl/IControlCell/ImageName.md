# Поле ImageName

## Описание

    ImageName :String;

## Назначение

Содержит название картинки (иконки), которая будет располагаться в ячейке.
Стандартные изображения находятся в папке *Attachment/Icons*.

Размер картинки определяется полем [ImageSize](ImageSize).
