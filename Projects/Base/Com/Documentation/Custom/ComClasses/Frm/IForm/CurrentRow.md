# Функция CurrentRow

## Описание

    func CurrentRow :Integer;

## Назначение

Возвращает номер текущей строки таблицы. Метод можно использовать в обработчиках, в которых нет
параметра с номером редактируемой строки, например OnGetLookupFilter.
