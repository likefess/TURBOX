[IContext](Default)

# Функция CreateEx

## Описание

    func CreateEx (aSender :IControlCell; aTable :Logical; aIndex :Integer) :IContext;

## Параметры

**aSender** - ячейка, для которой формируется контекст.

**aTable** - равно True, если ячейка лежит в таблице.

**aIndex** - индекс текущей строки.

## Назначение

Создание контекста.
