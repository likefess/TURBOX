#Persistent URL
---

В адресной строке браузера после стандартного URL можно указать какую конкретно форму
(и какую конкретно запись для бланков-редакторов) требуется открыть. Форму можно указать
как явно, например, Control.Rep.TasksList, так и через alias. Alias-ы добавляются в
расширениях метода Net.WebSrv.GetAlias. Например:

     #Override;
     func GetAlias :Storage;
       Result = extended GetAlias;
       Result['task'] = [Frm.Data.Task, 'PrintedNumber'];
     end;

Тут task - имя alias-а, Frm.Data.Task - класс формы, который вызовется,
PrintedNumber - поле, по которому будет искаться открываемая запись, Например URL

     http://localhost:81/erp/webui/index.html#/workspace/task/20.0001

открывает форму задачи, у которой поле PrintedNumber равно "20.0001".

Для не бланков-редакторов второй параметр указывать не нужно, например

     http://localhost:81/erp/webui/index.html#/workspace/Control.Rep.TasksList

откроет отчет "Список задач"

PURL можно использовать в уже открытой (авторизованной) веб-сессии, в этом случае
запрошенная форма откроется в новой цепочке окон, или в новом окне браузера, в этом
случае сначала откроется окно авторизации, а после авторизации восстановятся открытые
окна из прошлой веб-сессии и к ним добавятся запрашиваемая форма.

Есть 2 различия для случаев открытия в уже авторизованной сессии и в новом окне:

1. Если запрашиваемая форма уже есть на экране, то в случае нового окна браузера
после авторизации сессия запустится на уже существующей запрашиваемой форме, а в случае
авторизованной сессии всегда откроется ещё одно окно с запрашиваемой формой.

2. Если для бланка-редактора не указать второй параметр, то в авторизованной сессии
откроется редактор с вновь созданной записью, а в случае нового окна - запрашиваемая форма не откроется.

Если в качестве параметра указать несуществующую форму или несуществующую запись,
то возвращается ошибка "URL not found".

По понятным причинам в значении второго параметра не может быть использован символ '/',
поэтому задачу по номеру "2/99" открыть не получится.
