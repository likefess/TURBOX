# Интеграция с 1С файл.xml

-------

Из программы 1С выгружается файл в формате .xml

Порядок работы загрузки:

1.**Тип точки подключения**

Создаем тип точки подключения (Данные XML)

![](topic:Integration.AddFiles.Screenshot_10968.jpg)

2.**Точка подключения** 

Создаем точку подключения:

- Наименование - задается вручную название точки подключения  (XML)
- Тип точки подключения - выбираем из справочника Типы точек подключения (кнопка Выбор ![](topic:AddFiles.Buttons.Btn_select.png)). Карточка выбранного типа точки подключения доступна по кнопке  ![](topic:AddFiles.Buttons.Btn_go.png) (Данные XML)
- Полю FileName присваиваем Значение: кнопкой Выбор ![](topic:AddFiles.Buttons.Btn_select.png) из диалогового окна Выберите файл выделяем курсором нужный файл, подтверждаем кнопкой **Открыть**. (D:\TurboX\Users\localhost\Пример_BI_Otch\ex1.xml)

![](topic:Integration.AddFiles.Screenshot_10967.jpg)


3.**Настройка** 

Создаем Настройку:

- Наименование - задается вручную название настройки (импорт xml)
- Направление обмена - заполняется из выпадающего списка: загрузка 
- Вариант запуска - из выпадающего списка: по расписанию или вручную
- Точка подключения - выбираем из справочника Точек подключения (кнопка Выбор ![](topic:AddFiles.Buttons.Btn_select.png)). Карточка выбранной точки подключения доступна по кнопке  ![](topic:AddFiles.Buttons.Btn_go.png)> (XML)
- Правило сопоставления - заполняется из справочника Соответствие полей (кнопка Выбор ![](topic:AddFiles.Buttons.Btn_select.png)).Карточка выбранного правила доступна по кнопке  ![](topic:AddFiles.Buttons.Btn_go.png)
- Программный класс обработки - заполняется из выпадающего списка: Integration.Service.IRecursiveProcessor
- Обновление данных - выбрать из выпадающего списка: создать новый 
  
![](topic:Integration.AddFiles.Screenshot_10966.jpg)

4.Ручной запуск настройки обмена осуществляется по кнопке Сервисы ![](topic:AddFiles.Buttons.Btn_Services.png):

- Выполнить
- Выполнить асинхронно
