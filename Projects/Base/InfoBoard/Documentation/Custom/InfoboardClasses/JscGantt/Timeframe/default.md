# Ctrl.Jsc.Timeframe

Диапазон на оси времени.

Наследник класса [Ctrl.Jsc.Item](..\Item.default).
Отображается как вертикальная черта если задана только дата начала,
либо как область, если заданы обе даты.
Может показываться в легенде.
