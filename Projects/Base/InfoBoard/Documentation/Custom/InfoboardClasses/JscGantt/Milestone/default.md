# Ctrl.Jsc.Milestone

Метка на диаграмме.

Наследник класса [Ctrl.Jsc.Item](..\Milestones.default).
Метки привязываются к заданной [задаче](..\Task.default),
отображаются в виде [маркеров](..\Jsc.Marker)
и не имеют собственного имени.
