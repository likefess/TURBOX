Теперь, при любом сохранении карточки подразделения, для каждого установленного флага ("Это склад", "Это касса", 
"Это расч.счет") ищется процесс соответствующего типа (Склад, Касса, Расч.счет), и если ни одного соответствующего 
процесса для данного подразделения нет, то он создается.
Т.е. если для этого подразделения не было ни одного склада (кассы, расч.счета) и были установлены все три флага, то при сохранении создадутся три процесса соответствующих типов (Склад, Касса, Расч.счет).
При снятии флагов соответствующие процессы не удаляются.

     Если единица измерения упаковки задана, но количество товара в позиции меньше количества 
     товара в упаковке, то количество будет равно количеству товара.
     ВНИМАНИЕ!!! Описанное заполнение происходит только из системы "Торговля". 
     Если переформировать "Торг-12" из бухгалтерии, то поле очистится.

     Пункт меню располагается в УЦП\Запасы\Основные данные.
     Первоначальная реализация решает текущие потребности ТОРО по заказу и получению/возврату/списанию ТМЦ на складе.
     В документе имеется сервис «Отпустить ТМЦ по заявке», который формирует документ «Перемещение ТМЦ» со склада,
     удовлетворяющего заявку, на склад, получающий ТМЦ по этой заявке.

     В документе можно вызвать отчет "Состояние заявки", отражающий запрошенное, полученное и не полученное
     количество ТМЦ по текущей заявке.
