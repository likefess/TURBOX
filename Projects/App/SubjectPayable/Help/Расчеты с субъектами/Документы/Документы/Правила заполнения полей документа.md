#Правила заполнения полей документа «Заявка на оплату» 
____________________

В таблице приведены обязательные поля, которые должны быть заполнены, при автоматическом формировании документа
"Заявка на оплату" из документа "Начисление задолженности"
______________

| **Заявка на оплату**           | **Начисление задолженности**                                    | **Примечания**                                                                                                                                          |
|--------------------------------|---------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Номер                          |                                                         | Согласно нумератора документа «Заявка на оплату»                                                                                                                |
| Дата                           |                                                         | Текущая дата                                                                                                                                                    |
| Автор                          |                                                         | Учетная запись пользователя                                                                                                                                     |
| Ответственный                  |                                                         | Учетная запись пользователя                                                                                                                                     |
| Состояние                      |                                                         | Всегда = Новая заявка                                                                                                                                           |
| **Данные компании**            |                                                         |                                                                                                                                                                 |
| Предприятие                    | Предприятие                                             |                                                                                                                                                                 |
| Тип оплаты                     |                                                         | Всегда = Банк                                                                                                                                                   |
| Р/счет                         | Р\счет нашего предприятия                               | Из данных закладки «Интеграция документов»                                                                                                                           |
| ЦФО                            | ЦФО                                                     | Из данных закладки «Интеграция документов»                                                                                                                           |
| **Получатель и основание платежа** |                                                         |                                                                                                                                                             |
| Контрагент                     | Контрагент                                              |                                                                                                                                                                 |
| Расчетный счет                 | Р\счет контрагента                                      | Из данных закладки «Интеграция документов»                                                                                                                           |
| Срок оплаты                    | Срок оплаты                                             | Из данных закладки «Интеграция документов»                                                                                                                           |
| Прогнозный срок                |                                                         | Всегда равен Срок оплаты                                                                                                                                        |
| **Данные платежа**             |                                                         |                                                                                                                                                                 |
| Флаг Налоговый платеж          | Ставится только при Тип начисления = Расчеты по налогам |                                                                                                                                                                 |
| Сумма без НДС                  | Сумма по документу                                      | По умолчанию указывается сумма остатка задолженности(в валюте задолженности) на дату формирования «Заявки на оплату». Допускается корректировка пользователем. |
| Ставка НДС                     | Всегда «-»                                              |                                                                                                                                                                 |
| Сумма с НДС                    | Сумма по документу                                      | По умолчанию указывается сумма остатка задолженности(в валюте задолженности) на дату формирования «Заявки на оплату». Допускается корректировка пользователем. |
| Валюта платежа                 | Валюта суммы по документу                               | Валюта задолженности                                                                                                                                            |
| Очередность                    |                                                         | По умолчанию "5"                                                                                                                                                |
| **Налоговые реквизиты**        |                                                         |                                                                                                                                                                 |
| КБК                            | Налог                                                   | Заполняется  только при Тип начисления = Расчеты по налогам                                                                                                     |
