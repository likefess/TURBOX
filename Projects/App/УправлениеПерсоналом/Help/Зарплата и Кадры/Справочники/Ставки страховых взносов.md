#  Ставки страховых взносов
_ _ _ _ _ _

**Назначение справочника**: содержит законодательно утвержденные проценты отчислений по видам обязательного страхования, установленные НК РФ. Используется при расчете страховых взносов.

**Путь в системе**: Зарплата и Кадры/Справочники/Ставки/Ставки страховых взносов

**Порядок работы со справочником**:

1.Для добавления новой записи нажмите на кнопку "Добавить" ![](topic:Com.AddFiles.Buttons.Btn_Add.png) и заполните открывшийся бланк:

![](topic:.AddFiles.Screenshot_3124.jpg)

* Код - аналитический код записи. Заполняется вручную. Новый код не должен совпадать ни с одним из кодов ранее введенных записей.
* Название - наименование ставки СВ. Заполняется вручную.
* Дата начала - вручную устанавливается дата начала срока действия ставки СВ.
* Дата окончания - вручную устанавливается дата окончания срока действия ставки СВ.


***Закладка «Ставки»***:

* Единый страховой взнос (“<”) - тариф ставки Единого страхового взноса, устанавливаемого для доходов сотрудника, не превышающих единую предельную величину базы для исчисления страховых взносов. Заполняется вручную.
* Единый страховой взнос (“>”) - тариф ставки Единого страхового взноса, устанавливаемого для доходов сотрудника, превышающих единую предельную величину базы для исчисления страховых взносов. Заполняется вручную.
* ОМС (отд. категории) - тариф ставки Взноса на обязательное медицинское страхование в пределах установленной единой предельной величины базы для исчисления страховых взносов для отдельных категорий плательщиков (плательщиков страховых взносов, производящих выплаты и иные вознаграждения в пользу прокуроров, сотрудников Следственного комитета Российской Федерации, судей федеральных судов, мировых судей согласно с п. 4 ст.425 НК РФ)
* ФСС (отд. категории)- тариф ставки Взноса на обязательное социальное страхование на случай временной нетрудоспособности и в связи с материнством в пределах установленной единой предельной величины базы для исчисления страховых взносов для отдельных категорий плательщиков (плательщиков страховых взносов, производящих выплаты и иные вознаграждения в пользу прокуроров, сотрудников Следственного комитета Российской Федерации, судей федеральных судов, мировых судей согласно с п. 4 ст.425 НК РФ)
* ЕСВ (% свыше МРОТ)- тариф ставки Единого страхового взноса, устанавливаемого для доходов сотрудника в отношении части выплат в пользу физического лица, определяемой по итогам каждого календарного месяца как превышение над величиной минимального размера оплаты труда, установленного федеральным законом на начало расчетного периода (п 2.4 ст. 427 НК РФ).
* Замок - в установленном состоянии флаг означает, что документ закрыт для редактирования.
* Завершен - в установленном состоянии флаг означает, что документ закрыт по всем видам учета.


2.Для редактирования уже существующей записи воспользуйтесь кнопкой Редактировать ![](topic:Com.AddFiles.Buttons.Btn_Edit.png) .

3.Для сохранения введенной информации нажмите на кнопку Применить ![](topic:Com.AddFiles.Buttons.Btn_OK.png)  . Для сохранения документа и его закрытия нажмите кнопку ОК ![](topic:Com.AddFiles.Buttons.Btn_Post.png) , иначе - Отмена ![](topic:Com.AddFiles.Buttons.Btn_CloseCancel.png).
